# CV Interactif — Guy Morize

Site statique 4 pages, déployable sur Vercel en 5 minutes.

## Fichiers
- `index.html` — Page d'accueil
- `apropos.html` — À propos
- `parcours.html` — Parcours complet
- `contact.html` — Contact
- `style.css` — Styles
- `main.js` — Interactions

## Déploiement Vercel
1. Créer un compte sur github.com
2. Créer un nouveau repository "cv-morize"
3. Uploader tous les fichiers
4. Aller sur vercel.com → New Project → importer le repo GitHub
5. Cliquer Deploy — c'est tout !

## À personnaliser
- Remplacer l'URL LinkedIn dans index.html, apropos.html, contact.html
- Ajouter votre photo dans apropos.html (balise <img>)
- Ajouter votre vidéo dans index.html (balise <video>)
- Pour le formulaire contact : créer un compte sur formspree.io (gratuit)
